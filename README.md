# IDS-ML_1.0
🛡️ Network IDS with ML - Ensemble classification (RF, XGBoost, NN) + anomaly detection. Stack: FastAPI backend, React frontend, PostgreSQL OLTP, ClickHouse OLAP, Redis caching, Kubernetes orchestration, Prometheus metrics, ELK logging, automated retraining. Features real-time API predictions, interactive dashboards, horizontal scaling.
